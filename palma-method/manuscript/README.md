## Chapter drafting convention

Each chapter should contain, in this order where useful:

1. Opening scene or managerial problem.
2. Core distinction or argument.
3. Evidence and mechanism.
4. Case, live observation, or field note.
5. Counterargument or limitation.
6. Practical implication.
7. Transition to the next chapter.
