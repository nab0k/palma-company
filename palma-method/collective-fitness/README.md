# The Collective Fitness — Live Action Research

This folder documents how the Palma Method begins in a real social environment in Portugal. It is not evidence of a complete organizational transformation.

## Research purpose

- Observe what is already alive.
- Understand why people return and where interaction stops.
- Identify roles, rituals, subgroups, and bridges.
- Test one or two small, ethical interventions.
- Document response, uncertainty, failed assumptions, and changes in strategy.

## Data handling

- Use participant labels and anonymize by default.
- Store raw recordings only in ignored private folders.
- Obtain explicit permission for names, exact quotes, photos, and identifiable stories.
- Keep observation separate from interpretation.
- Do not expose private commercial information.
