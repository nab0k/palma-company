# SPEC_PALMA_KNOWLEDGE_OS_0001

## Purpose
Palma Knowledge OS is the operational system for the Palma Method.

## Principles
1. Notion is the operational workspace.
2. GitHub is the canonical repository.
3. Every long-lived artifact lives in GitHub.
4. Every active workflow is visible in Notion.

## Canonical ownership
- Current drafts: Notion
- Accepted drafts: GitHub
- Research cards: GitHub
- Tasks: Notion
- Decisions: GitHub (mirrored)
- Project status: Project Ledger
- Event history: GitHub
